    • mac上搭建hexo+github page

    [TOC]

    1. 安装node

    brew install node

    2. 安装hexo

    npm install -g hexo-cli

    3. 初始化hexo项目

    hexo init hexoDir
    cd hexoDir
    npm install

    4. 关联github page

    1. 创建github项目，名字前缀和自己的用户名一样： fkcmd.github.io
    2. 修改hexoDir/_config.xml

    1. 安装

      npm install hexo-deployer-git –save

    2. 更新项目，push到github

      hexo clean
      hexo generate
      hexo server(这一步为了在local检验，课省略)
      hexo deploy

    5. 遇到的问题

    1. hexo clean/g/s 时出错：
    • _config.yml 中配置有问题 ： #Site 下的配置，：后面的属性要有空格间隔
    1. hexo d时 ：
    • 没有安装hexo-deployer-git

      npm install hexo-deployer-git –save
